📋 How to get your OAuth Client ID (click to expand)
  1. Go to console.cloud.google.com and create or select a project
  2. Go to APIs & Services → Library and enable the Google Sheets API
  3. Go to APIs & Services → OAuth consent screen → choose External → fill in app name (e.g. "Vital") → save
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth client ID
  5. Choose Web application, then under Authorized JavaScript origins add the URL you're opening this file from (e.g. https://your-domain.com or if running locally: http://localhost or the Google Drive viewer URL)
  6. Copy the Client ID (ends in .apps.googleusercontent.com) and paste below
  7. ⚠️ If running from Google Drive: use https://drive.google.com as the authorized origin
